Minolta Dimage Scan Dual III / IV

VueScan · macOS M1 · USB — Scanworkflow

01
Aansluiten
  • Gebruik een korte USB-kabel — de originele kabel is van lage kwaliteit
  • Scanner aan — wacht ~5 minuten voor de eerste scan zodat de lamp stabiliseert
  • VueScan openen
De scanner gebruikt een CCFL-lamp die op temperatuur moet komen. Sla je de opwarmtijd over, dan kunnen de eerste scans iets afwijken in belichting.
02
Input tab
  • Media → Color Negative
  • Bit depth → 16-bit
  • Resolution → 3200 dpi
  • Batch scan → aan
  • Auto-advance → aan
03
Filter tab
  • Infrared Clean → uit (scanner heeft geen infraroodkanaal)
  • Restore Fading → Light bij oud materiaal, anders uit
  • Grain reduction → Light of Medium bij geduwd of hooggevoelig materiaal (ISO 1600+), anders uit
  • Sharpen → uit — doe dit achteraf in Lightroom of Capture One
04
Color tab
  • Auto levels → aan
  • Negative vendor → Generic
  • Negative brand → Generic
  • Negative type → Color Negative
05
Lock Exposure
  • Leg een leeg stukje filmrand (tussen frames of buiten het beeldvlak) in de scanner
  • Doe een preview — VueScan kalibreert de belichting op de oranje filmbase
  • Klik op Lock Exposure in de Input tab
  • Verwijder de filmrand, laad het eerste frame en begin met scannen
Zonder vergrendelde belichting herberekent VueScan per frame, wat kleurverschillen tussen opnames van dezelfde rol geeft. Lock Exposure houdt de belichting constant over de hele rol.
06
Output tab
  • Raw file → aan
  • Raw file type → DNG
  • Raw compression → aan (lossless, scheelt schijfruimte)
  • JPEG file → aan
  • JPEG quality → 95
  • JPEG file name → scan0001+.jpg
07
Scannen
  • Negatief schoonblazen met luchtspuitje
  • Preview checken in VueScan
  • Batch scan starten — VueScan loopt automatisch door alle frames
Multi-sampling: 2× is een goed compromis voor kritieke beelden — vermindert ruis in de schaduwen merkbaar. Hoger dan 4× geeft nauwelijks extra winst maar verdubbelt de scantijd.

07
Batch desaturatie — ImageMagick
  • Open Terminal
  • Navigeer naar je scanmap: typ cd en sleep de map vanuit Finder naar Terminal
  • Druk op Enter, voer dan het commando uit:
mkdir bw for f in *.jpg; do magick "$f" -colorspace Gray "bw/$f"; done
De originelen blijven ongewijzigd. De zwart-wit versies komen in de submap bw/. Vervang *.jpg door *.tif als je TIFF-bestanden hebt.
